# Rareș Bighiu

Mathematics and Computer Science student at Alexandru Ioan Cuza University in Iași. Most of my time goes into security tooling: programs that find secrets, watch network traffic, or break things in a controlled way so I understand how the defense should work.

## What I'm building

- [ShotShield](https://github.com/Raresney/ShotShield) spots sensitive data in a screenshot (API keys, IBANs, cards, IDs, even faces) and lets you cover it before you share. Everything runs on your machine. There's a Windows installer on the releases page.
- [PacketHighway](https://github.com/Raresney/PacketHighway) turns live network traffic into a 3D city where each packet is a vehicle on the highway. Built with scapy and React Three Fiber.
- [AI-Security-Research](https://github.com/Raresney/AI-Security-Research) is where I poke at language models: prompt injection, phishing classification, and network recon.
- [Cyber-lab](https://github.com/Raresney/Cyber-lab) holds the smaller experiments, mostly Python tooling and network analysis.

## Tools I reach for

Python, TypeScript, Rust (through Tauri), Linux, networking, and the usual machine learning basics.
